Рівняння: \[2 - x = ln(x)\]

Рівняння не має простого аналітичного розв'язку через комбінацію поліноміальних і логарифмічних функцій. Для знаходження його коренів застосовуються чисельні методи.


Код Scilab:

1.2.sce
//Визначаємо функцію
function y = myFunction(x)
y = 2 - x - log(x);
endfunction

x0 = 1;

//Розв'язок функції через x0
x_solution = fsolve(x0, myFunction);

//Виводимо результат
disp("Відповідь: ");
disp(x_solution);

ВАРІАНТ 9

Система рівнянь:

\[ \begin{cases} x = x^2 + y^2 - 0.1 \\ y = 2xy - 0.1 \end{cases} \]

Код Scilab:

1.3.sce
// Система рівнянь
function [x_next, y_next] = simple_iteration(x, y)
x_next = x^2 + y^2 - 0.1;
y_next = 2*x*y - 0.1;
endfunction

// Розв'язання за допомогою методу ітерації
function [x_sol, y_sol] = simple_iterative_method(x0, y0, max_iter, tol)

//Початкова точка (0;0)
x = x0;
y = y0;

//Цикл ітерацій
for iter = 1:max_iter
[x_next, y_next] = simple_iteration(x, y); //визначення x та y
if norm([x_next - x; y_next - y]) < tol //якщо норма вектора змін менша за точність
break;
end
x = x_next;
y = y_next;
end

x_sol = x;
y_sol = y;
endfunction

max_iterations = 100; //Кількість ітерації (за замовчуванням)
tolerance = 1e-6; //Точність

//Розв'язок системи рівнянь
[x_solution, y_solution] = simple_iterative_method(0, 0, max_iterations, tolerance);

disp(['Відповідь:']);
disp(['x = ', string(x_solution)]);
disp(['y = ', string(y_solution)]);
